### Requirements
* Ansible
* Docker
* Make

### Setup servers

```
$ make install-requirements  - setup requirements  
```
```
$ make install-docker  - install docker  
```
```
$ make install  - install pip, (.env) file
```
```
$ deploy - Start Redmine
```
```
$ install-datadog - install and setup Datadog
```
